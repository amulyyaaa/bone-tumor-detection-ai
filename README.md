🧠 AI-Based Medical Image Tumor Detection System

An end-to-end deep learning web application for AI-assisted tumor detection using medical imaging and computer vision techniques. This project uses a Convolutional Neural Network (CNN) with TensorFlow/Keras to classify uploaded medical scan images and provide prediction results with confidence scores and Grad-CAM visualization.

The application is deployed as an interactive web app using Streamlit and Render, allowing users to upload images and receive real-time AI-generated analysis.

 🚀 Features

* Upload medical scan images through a web interface
* AI-powered tumor classification
* Confidence score prediction
* Grad-CAM attention heatmap visualization
* Interactive and responsive Streamlit UI
* Cloud deployment using Render
* End-to-end machine learning workflow

## 📊 Model Details

* Model Architecture: MobileNetV2 Transfer Learning
* Input Size: 224 × 224 RGB Images
* Framework: TensorFlow/Keras
* Output Classes:

  * Tumor Detected
  * No Tumor

The model was trained on a labeled medical imaging dataset and integrated into a real-time prediction pipeline.


## 🔥 Explainable AI with Grad-CAM

The project includes Grad-CAM (Gradient-weighted Class Activation Mapping) visualization to highlight the regions of the image where the model focuses while making predictions.

This improves interpretability and helps visualize the decision-making behavior of the CNN model.


## 🌐 Deployment
